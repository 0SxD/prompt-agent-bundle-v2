# prompt-agent-bundle-v2

v3 governance bundle for the PMAO (Prompt-engineering Multi-Agent Orchestration) agent OS.

## Status

R&D / Experimental. Maintained by Sage / 0SxD as part of an ongoing prompt-engineering and agent-skills research portfolio.

## What this is

The second generation of an agent governance scaffold built around the Trinity Dialectic (Pathos, Ethos, Logos) 15-point scoring rubric. The v3 bundle tightens governance across all six files: bootstrap, system directive, canonical spec, skills ledger, metrics tracker, and issue log. Skills are seeded but left as PENDING_DRAFT by design: each skill only gets drafted after a real gate-fire or hard-fail, keeping the bundle honest and bloat-free.

## Layout

- `bootstrap.md` -- session boot protocol and L1 BOOT sequence
- `system_directive.md` -- agent OS system directive (v3)
- `spec.md` -- HOT_CACHE / GRAPH_LINKS / COMPACTION_LEDGER (canonical state)
- `skills.md` -- 10 seeded skills, PENDING_DRAFT
- `metrics_ledger.md` -- cost and token telemetry
- `issues.md` -- open confusion and illegibility tracking
- `LICENSE` -- MIT
